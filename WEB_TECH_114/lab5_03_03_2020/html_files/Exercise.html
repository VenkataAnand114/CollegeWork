Home Profile Contact
In general, the home tab is a tab or button in an application or web page that returns you to the home section. A home page, homepage or a start page is the initial or main web page of a website or a browser. The initial page of a website is sometimes called main page as well.A home page is a webpage that serves as the starting point of website. It is the default webpage that loads when you visit a web address that only contains a domain name.
A profile page is a Scratcher's user page. It can be accessed by clicking on the "profile page" link in the dropdown on the toolbar if a Scratcher is logged in, or by clicking on the Scratcher's username next to any project, comment, or forum post by them.
A contact page is a standard web page on a website used to allow the visitor to contact the website owner or people who are responsible for the maintenance of the site. The page often contains one or more of the following items: a mailto link to an e-mail address. a description of personalia like an (organization) name, address, zip code, residential area, with sometimes a map indicating a certain physical location and a description or visualization of how to get there; a set of visual icons or textual links to social networking services; a contact form with entries where the visitor can fill in their name, subject and message and send or reset it.
